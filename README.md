# Elite 102 Banking App 🏦

## Overview

This Banking App allows users to:

- Create new bank accounts (name + initial deposit)
- Deposit money into accounts
- Withdraw money (with balance validation)
- Check account balances
- List and manage existing accounts
  
## Setup Instructions
- Python 3.12
- VS Code
- SQLite
- GitHub for version control
  
## How to run
- Make sure Python 3 is installed
- Clone or download this project
- Open a terminal in the project folder
- Run: python main.py
  
## Extensions completed
- Wire transfers between accounts
- Transaction history log with timestamps
- Advanced UI (colors, formatted tables, input validation)
- Interest calculation
- Overdraft protection
